Détail Projet

Traiter des données.

Date : 01/2026

🎯 Contexte et Objectifs

Dans le cadre de ma formation, j'ai réalisé la SAÉ 1.05 dédiée au traitement de données avec le langage Python. L'infrastructure d'un technicien RT demandant de savoir manipuler des scripts et des flux d'informations, l'objectif de ce projet était de récupérer des données géographiques réelles via les API de la plateforme collaborative OpenStreetMap. Il s'agissait de développer un outil capable d'extraire automatiquement des informations sur des points d'intérêt (comme les restaurants de Caen) afin de générer des fiches de synthèse et de calculer des statistiques locales aux formats Markdown et HTML.

🛠 Apprentissages & Réalisations

  • AC 11.03 : Programmer des applications simples.
  • AC 11.05 : Traiter des données.

  • Requêtes API : Utilisation des API REST d'OpenStreetMap et d'Overpass en Python pour télécharger des fichiers de données au format JSON.
  • Développement de scripts : Création de fonctions modulaires pour extraire automatiquement des attributs spécifiques d'un lieu (nom, horaires, coordonnées).
  • Calculs statistiques : Programmation d'algorithmes pour analyser un jeu de données locales et calculer des proportions (comme le pourcentage d'établissements acceptant la carte bancaire ou proposant la vente à emporter).
  • Génération de documents : Écriture d'une fonction de conversion automatisée pour transformer des fichiers texte structurés en pages HTML valides (via la bibliothèque markdown).
  • Documentation du code : Rédaction de commentaires et de documentations claires pour expliquer la logique et le fonctionnement des fonctions de calcul.
