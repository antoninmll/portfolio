Détail Projet

Se présenter sur internet.

Date : 12/2025

🎯 Contexte et Objectifs

Dans le cadre du BUT Réseaux et Télécommunications, la SAÉ 1.04 ("Se présenter sur Internet") consistait à créer un site web vitrine multipages en HTML/CSS sur un thème libre. L'objectif principal était de concevoir un site moderne, entièrement adaptatif (responsive) et accessible (normes WCAG). Ce projet visait à nous faire maîtriser les bases du développement web (Flexbox, Grid, animations) et l'utilisation d'outils professionnels comme Git et GitHub pour la gestion de version et l'hébergement en ligne.

🛠 Apprentissages & Réalisations

  • AC 13.01 : Maîtriser les technologies de base du Web (HTML/CSS) pour structurer et styliser une application.
  • AC 13.02 : Utiliser des outils de gestion de version et déployer un site en ligne.

  • Développement multipages : Création et structuration d'un site web complet (Accueil, Thème, À propos) en utilisant du HTML5 propre et sémantique.
  • Mise en page moderne : Utilisation de CSS Grid et Flexbox pour concevoir une interface soignée avec une charte graphique cohérente et des animations graphiques.
  • Responsive Web Design : Intégration de règles adaptatives (Media Queries) pour que le site s'affiche parfaitement sur ordinateur, tablette et mobile.
  • Qualité et Accessibilité : Optimisation du code pour passer la validation W3C et respecter la norme d'accessibilité WCAG 2.0 AA.
  • Gestion de version avec Git : Utilisation d'un dépôt GitHub avec un historique de commits réguliers, clairs et professionnels pour suivre l'évolution du projet.
  • Déploiement et Documentation : Rédaction d'un fichier README.md détaillé et mise en ligne du site via GitHub Pages.
